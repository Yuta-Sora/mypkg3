# mypkg3
このパッケージは、ROS上でパブリッシャから入力した文字をサブスクライバ側で表示できるプログラムです。  
また、パブリッシャ側で打ち込んだ２つの数字の和、または差を求めることもできます。
## 機能紹介    
このプログラムには5つの機能があります。
パブリッシャ側ノード:count.py  
サブスクライバ側ノード:twice.py  
機能1:count.py側で入力した文字をtwice.py側で表示する
機能2:count.py側で <+> を入力すると
